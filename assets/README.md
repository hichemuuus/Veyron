Place actual screenshots and diagrams here before the final release.

## Generating Screenshots

1. Launch Veyron in production mode: `npm run tauri:build` → run the installer
2. Capture the full window with the dashboard visible
3. Save as `veyron-screenshot.png` (1920×1080 recommended)
4. For the demo GIF, record 15-30s of task creation and execution
